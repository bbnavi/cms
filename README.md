# BBNAVI-CMS

BBNAVI-CMS is a simple CMS based on BBNAVI Datahub API to manage the content within the BBNAVI platform, imported through the BBNAVI Datahub API by Dataproviders.

## Installation

```bash
$ npm install
```

or using yarn

```bash
$ yarn install
```


### Requirements

- Node 16.13 or higher

## Usage

```bash
$ npm run dev
```

or using yarn

```bash
$ yarn dev
```
